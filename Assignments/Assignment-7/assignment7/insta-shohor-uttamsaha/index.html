2.Difference between Local Storage & Session Storage:

    Local Storage:
  • Local Storage store the data permanently that doesn't expire. For example: Suppose I visited an E-commerce website and added some items in the carts but didn't buy and if I go to the webpage after few months I can see the item's that i already added before to the cart.
  • The data stored in localStorage persists until explicitly deleted. The Changes are saved and available for all current and future visits to the webpage.
    • Session Storage:
  • In the Session Storage it store data temporarily. The data in sessionStorage is cleared when the page session is end
  • The data is available only inside the window/tab in which it was set.
  • Once the window is closed, the storage will be deleted automatically.
